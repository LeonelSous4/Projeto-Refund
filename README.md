Este é um projeto prático de desenvolvimento web focado na criação de um sistema de controle de despesas e solicitação de reembolso. O objetivo foi aplicar conceitos fundamentais de JavaScript, manipulação do DOM e construção de interfaces modernas.

🚀 [Clique aqui para testar o projeto ao vivo!](https://leonelsous4.github.io/Projeto-Refund/)

---

## 💻 Sobre o Projeto

A aplicação permite adicionar, visualizar e remover despesas, com atualização automática do total e da quantidade de itens.

### Principais funcionalidades:
- Adição de despesas com nome, categoria e valor
- Máscara automática de moeda (R$) durante a digitação
- Listagem dinâmica das despesas
- Remoção de itens da lista
- Cálculo automático do total de gastos
- Contador de quantidade de despesas
- Interface moderna e responsiva

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura da aplicação
- **CSS3:** Estilização, responsividade e efeitos visuais (glassmorphism)
- **JavaScript (ES6+):**
  - Manipulação do DOM
  - Eventos (oninput, onsubmit, click)
  - Criação dinâmica de elementos
  - Formatação de moeda com `toLocaleString`
